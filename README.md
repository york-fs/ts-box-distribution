The TS box distribution board lives in the high voltage TS box enclosure and supply LVS power and signals to the
energy meter, discharge relay, and the interlock of the high voltage disconnect.

A recursive clone must be used to download the `kicad-library` repository:

    git clone --recursive https://github.com/york-fs/ts-box-distribution.git
